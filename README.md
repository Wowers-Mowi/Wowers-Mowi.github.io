# Plan de vuelo

App local para rellenar un plan de vuelo paso a paso y copiar la previsualizacion como imagen PNG al portapapeles.

Incluye un menu `Opciones` para personalizar y ordenar los desplegables. En las preguntas escritas, al guardar opciones desde ese menu, la pregunta permite alternar entre escribir manualmente o elegir desde un desplegable. Los cambios del menu se aplican al pulsar `Guardar`.

La copia de imagen al portapapeles requiere un navegador compatible, como Edge o Chrome. Si el navegador bloquea la copia, el enlace `Descargar PNG` guarda la misma imagen.
